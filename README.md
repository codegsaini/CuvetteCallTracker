# CuvetteCallTracker

### Objectives achieved

- [x] Total calls
- [x] Number of incoming calls
- [x] Number of outgoing calls
- [x] Number of missed calls
- [x] Duration of each call
- [x] Total call duration
- [x] Phone numbers involved in the calls
